# 🎓 Online Course Management System (FastAPI)

## 📌 Project Overview

This project is a backend API system built using FastAPI for managing an online learning platform.

Users can:

* Browse courses
* Enroll in courses
* Manage wishlist
* Search, filter, and paginate courses

---

##  Features

* REST APIs using FastAPI
* Pydantic validation
* CRUD operations
* Multi-step workflow (Wishlist → Enrollment)
* Search, sorting, and pagination
* Swagger API testing

---

##  Tech Stack

* Python
* FastAPI
* Uvicorn

---

## ▶ 
How to Run

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

Open:
http://127.0.0.1:8000/docs
